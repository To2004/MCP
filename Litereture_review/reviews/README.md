# Reviews — Generated Analysis Documents

This folder contains AI-generated review deliverables that synthesize findings from the literature review papers.

## Files

| File | Description |
|------|-------------|
| `datasets_review.md` | Markdown review of datasets used across the surveyed papers |
| `datasets_review.docx` | Word document version of the datasets review |
| `datasets_review.xlsx` | Spreadsheet version with structured dataset information |
| `benchmarks_review.md` | Markdown review of benchmarks and evaluation frameworks |
| `benchmarks_review.docx` | Word document version of the benchmarks review |
| `benchmarks_review.xlsx` | Spreadsheet version with structured benchmark information |
| `02_MCPShield_summary_HE.md` | Hebrew deep-dive summary of MCPShield paper with metaphors, code, and diagrams |

## Subfolders

| Folder | Description |
|--------|-------------|
| `paper_summaries/` | Detailed Hebrew summaries of individual papers with technical analysis and thesis relevance |
| `benchmark_md/` | Individual benchmark analysis files with data structure, risk dimensions, and usefulness verdicts for 1-10 risk scoring |

## How to Use

- Read the `.md` files for quick reference — they are plain-text readable
- Use the `.docx` files for formatted documents suitable for submission or sharing
- Use the `.xlsx` files for structured data that can be filtered, sorted, or integrated into other spreadsheets
- Each review topic (datasets, benchmarks) has all three formats with identical content
