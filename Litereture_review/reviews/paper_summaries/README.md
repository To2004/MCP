# Paper Summaries — Detailed Hebrew Summaries

This folder contains comprehensive Hebrew-language summaries of individual research papers from the MCP security literature review. Each summary provides deep technical analysis with metaphors, code examples, ASCII diagrams, and relevance to the MCP Dynamic Risk Scoring thesis.

## Files

| File | Paper | Key Topics |
|------|-------|------------|
| `02_MCPShield_summary_HE.md` | MCPShield (Zhou et al., 2026) | Server-side cognitive security, lifecycle defense, trust calibration |
| `09_Breaking_the_Protocol_summary_HE.md` | Breaking the Protocol (Maloyan & Namiot, 2026) | Protocol-level vulnerabilities, capability attestation, PROTOAMP benchmark |

## Summary Format

Each summary follows a consistent structure:

1. **Title and metadata** — Hebrew title, authors, venue, arXiv ID
2. **Core problem** — What is broken and why, with metaphors
3. **Methodology** — Technical approach, frameworks, experimental setup
4. **Key contributions** — Main findings with tables and diagrams
5. **Technical details** — Formulas, architectures, algorithms, code blocks
6. **Results** — Quantitative findings with comparison tables
7. **Limitations** — Critical analysis of weaknesses and gaps
8. **Relevance to thesis** — How the paper connects to MCP-RSS dynamic risk scoring
