# MCP Security Documentation

Welcome to the MCP Security project documentation. This folder contains
comprehensive guides, standards, and references for working with the project.

## Conventions

- All docs are Markdown files
- Each file covers one topic
- Files are kept concise — prefer short sections over long walls of text
- Code examples use the project's actual conventions
- Links are relative so they work on GitHub and locally
