# Weekly Paper Reports

This folder contains weekly reports generated every Saturday by Claude Cowork.

Each report is named `YYYY-MM-DD_weekly_papers.md` and covers new papers
published in the past 7 days related to MCP security, agent attacks, and
LLM tool security.

## Format

Each report includes:
- Paper title, authors, date, and source
- 3-sentence summary
- Relevance to defending MCP servers from malicious agents
- Relevance score (1–5)
- Key Takeaways section at the end
